Class tui-container makes a container inside content
Class tui-row and set of tui-col_* classes make a grid inside a container

Container classes

  • .tui-container: width 1104px on screens >= 1360px and 824px on screens <= 1359px
  • .tui-container_adaptive: width 1104px on screens >= 1360px, 824px on screens >= 1024px and <= 1359px, 608px on screens >= 768px and <= 1023px, 512px on screens >= 600px and <= 767px, 100% on screens < 600px
  • .tui-container_fullwidth: width 100% with 24px margins on screens >= 1024px and 16px on screens <= 1023px
  • .tui-container_fixed: width 1104px

Columns classes for adaptive layout

  • .tui-col_lg-*: screen width >= 1360px (Desktop L)
  • .tui-col_md-*: 1024px =< screen width <= 1359px (Desktop M)
  • .tui-col_sm-*: 600px =< screen width <= 1023px (Tablet)
  • .tui-col_s-*: 600px =< screen width <= 767px (Tablet S)
  • .tui-col_xs-*: 320px =< screen width <= 599px (Mobile)
