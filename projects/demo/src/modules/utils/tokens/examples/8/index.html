Using TUI_NUMBER_FORMAT injection token you can customize numbers formatting.
For example: 10 500,33
Can be customized as: 10/500.33

Defaults:

  • decimalSeparator = ,
  • thousandSeparator = CHAR_NO_BREAK_SPACE
  • zeroPadding = true
  • = negative-only

    Available options:

    • negative-only — shows sign for negative only (by default)
    • always — always shows sign (except 0)
    • never — never shows sign
    • force-negative — always shows minus (except 0)
    • force-positive — always shows plus (except 0)

Components that are customizable:

  • TuiMoneyComponent
  • TuiFormatNumberPipe
  • TuiInputNumberComponent
  • TuiInputSliderComponent
  • TuiInputRangeComponent
