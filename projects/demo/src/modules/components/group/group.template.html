Компонент и директика для группировки других компонентов, например, полей ввода и кнопок.
Элементы внутри группы одинаково растягиваются на максимально возможную ширину. Чтобы элемент не растягивался, есть класс class="tui-group__auto-width-item".
Чтобы элемент мог наследовать скругление, есть класс class="tui-group__inherit-item". Пригодится в группе инпутов с выводом ошибок.
