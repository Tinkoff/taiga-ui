Абстрактный класс AbstractTuiThemeSwitcher используется для создания компонентов для динамического переключения тем. В примере выше показан такой компонент — он наследуется от AbstractTuiThemeSwitcher и содержит переопределения переменных стилей. Для динамического включения/выключения можно использовать *ngIf. Важно: компонент должен иметь выключенную инкапсуляцию стилей.
