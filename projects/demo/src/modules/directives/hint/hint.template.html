Директива для показа подсказки при наведении на произвольный элемент. Наведи! Содержимое тултипа Для соответствия Accessibility требованиям, подсказка должна ставиться в соответствие фокусируемому элементу и пользователи со Screen Reader`ами должны слышать текст подсказки при фокусе после небольшой задержки. Для реализации, задайте уникальный tuiHintId и повесьте директиву [tuiDescribedBy]="тот же id" на фокусируемый элемент.
