Директива позволяет кастомизировать TuiPrimitiveTextfield, передавая параметры в него через механизм DI.

Это значит, что директива может находиться на несколько элементов выше, но все равно будет поставлять данные в дочерний textfield.

Если над одним текстфилдом будет несколько контроллеров, то их параметры смержатся. Приоритет будет отдаваться контроллерам, которые ближе к текстфилду

  2. Use it on tui-primitive-textfield or its parent elements
