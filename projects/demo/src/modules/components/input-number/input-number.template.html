InputNumber — компонент для ввода чисел. Введенное значение форматируется: тысячи разделяются пробелом. В отличие от Input значением поля является число, а не строка.
Компонент также может использоваться для ввода денежных сумм, т.к. в качестве постфикса можно использовать знак валюты с помощью tuiCurrency пайпа. Другие компоненты для ввода чисел: InputCount для ввода целых чисел не больше 999, Slider и InputSlider для ввода с помощью ползунка.
