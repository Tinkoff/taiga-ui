Теги предназначены для выводы информации в виде тегов. Их можно удалять, редактировать, при этом можно подписаться на эти действия. Обладает обширным набором модификаций внешнего вида.
Селектором для тега может быть как стандартный tui-tag, так и a[tuiTag]. Автоматическое задание цвета для тега Компонент задизейблен Тег можно редактировать Подписка на изменение тега (если тег отредактирован до пробелов/пустой строки — эмиттится событие удаления) Позволяет включить эффект наведения (hover). Контент слева от текста тега Ограничение на длину тега (при редактировании) Компонент можно удалять Показать лоадер Размер тега Статус меняет цвет тега Строка для вывода Подписка на редактирование тега, эмитит пустую строку при удалении
