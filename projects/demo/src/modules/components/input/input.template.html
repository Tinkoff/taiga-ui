Input — однострочное текстовое поле. Не рекомендуется использовать для ввода чисел, т.к. значением является строка.
Для ввода данных других типов используйте:
  • TextArea — для ввода многострочного текста,
  • InputDate — для ввода дат,
  • InputDateRange — для ввода периода дат,
  • InputNumber — для ввода чисел (также поддерживает меру измерения),
  • InputPassword — для ввода паролей,
  • InputPhone — для ввода телефонных номеров,
  • InputTag — для ввода набора тэгов,
  • InputRange, InputSlider — для ввода точных числовых значений,
  • Slider, Range — для ввода неточных числовых значений,
  • InputCount — для ввода количества.
