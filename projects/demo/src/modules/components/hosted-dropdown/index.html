This component is deprecated, use [(tuiDropdownOpen)] instead

tui-hosted-dropdown is a wrapper for focusable elements to attach to them a dropdown. Working with keyboard is also included.

If a wrapped element is input, arrow down press opens a dropdown. The next press focuses the first item from the list.

If it is not, mouseclick opens and closes a dropdown. A component uses the first focusable item inside component as a host for opening a dropdown. If you want another element to be host, use tuiHostedDropdownHost directive.
