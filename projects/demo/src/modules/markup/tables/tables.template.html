Таблицы создаются блоком .tui-table и его элементами и модификациями:

Элементы

  • .tui-table__tr: строка таблицы
  • .tui-table__th: заголовочная ячейка
  • .tui-table__td: ячейка
  • .tui-table__subtext: маленький и серый текст в ячейке
  • .tui-table__sort: кнопка сортировки
  • .tui-table__sort-icon: стрелка в кнопке сортировки

Модификаторы

  • .tui-table_layout_fixed: Фиксированная ширина ячеек
  • .tui-table_font-size_l: l-размер текста для всей таблицы
  • .tui-table_font-size_s: s-размер текста для всей таблицы
  • .tui-table_size_l: l-размер ячеек для всей таблицы
  • .tui-table__tr_border_top: верхний бордер для ряда
  • .tui-table__tr_border_none: отключить нижний бордер
  • .tui-table__tr_hover_disabled: отключить смену фона при наведении
  • .tui-table__tr_cursor_pointer: включить cursor: pointer при наведении
  • .tui-table__th_font-size_l: l-размер текста для заголовочной ячейки
  • .tui-table__td_font-size_l: l-размер текста для ячейки
  • .tui-table__td_font-size_s: s-размер текста для ячейки
  • .tui-table__td_size_l: l-размер для ячейки
  • .tui-table__td_align_center: вертикальное выравнивание по центру в ячеейке
  • .tui-table__td_first, .tui-table__th_first: нулевой паддинг слева в ячейке
  • .tui-table__td_last, .tui-table__th_last: нулевой паддинг справа в ячейке
  • .tui-table__td_text_center, .tui-table__th_text_center: текст по центру в ячейке
  • .tui-table__td_text_right, .tui-table__th_text_right: текст справа в ячейке
  • .tui-table__td_text_overflow: однострочное обрезание текста с многоточием в ячейке
  • .tui-table__sort_active: активное состояние сортировочной кнопки
  • .tui-table__sort_up: состояние сортировочной кнопки, при котором стрелочка повернута вверх
