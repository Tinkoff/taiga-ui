tui-hosted-dropdown — обёртка для фокусируемых элементов для прикрепления к ним выпадашки с заложенной работой с клавиатуры. Если обёрнутый элемент — инпут, то выпадашка открывается по нажатию стрелки вниз, последующее нажатие стрелки вниз переводит фокус на первый фокусируемый элемент выпадашки. Если обёрнутый элемент — любой другой, к примеру, кнопка, то по клику мышки по нему выпадашка будет открываться и закрываться. По умолчанию, в качестве "хоста" используется первый фокусируемый элемент внутри компонента, если нужно задать область поиска фокусируемого компонента более точечно — можно пометить необходимый тэг с помощью директивы tuiHostedDropdownHost.
