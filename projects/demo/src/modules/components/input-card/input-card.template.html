Для валидации ввода номера карты алгоритмом Луна добавьте tuiCreateLuhnValidator(customMessage?) к валидаторам контрола.

При [tuiTextfieldLabelOutside]="true" и фокусировании в поле в InputCard и InputExpire в поле показывается exampleText, а в InputCVC — placeholder.
