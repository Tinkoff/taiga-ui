 Содержимое выпадашки Опорная сторона, от которой строится контент выпадашки. При sided — сторона выпадашки, которая "прилипает" к хосту, иначе, при tuiDropdownLimitWidth !== 'fixed' — сторона хоста, по которой выпадашка выровнена. При tuiDropdownLimitWidth === 'fixed' не имеет эффекта, так как и левая и правая стороны выпадашки фиксированы по ширине хоста. Задать явно направление открытия выпадашки Задать явно элемент, считающийся инициатором выпадашки — при выходе фокуса из выпадашки через Tab/Shift+Tab этот элемент будет учитываться для поиска нового элемента для фокуса. По умолчанию используется элемент, на котором находится директива. Ограничить выпадающий список по ширине Минимальная высота выпадающего списка, при которой учитывается принудительное направление открытия Максимальная высота выпадающего списка Селектор директивы.
