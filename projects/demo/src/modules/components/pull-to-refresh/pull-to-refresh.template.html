Компонент для обновления контента блока по пуллу сверху вниз. Эмулирует внешний вид и поведение нативных вариантов из iOS и Android.
Компонент эмитит событие (pulled), когда пользователь инициировал загрузку.
Завершить загрузку можно стримом TUI_LOADED, который задается через DI
