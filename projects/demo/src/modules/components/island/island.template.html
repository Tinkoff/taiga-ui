Острова (айленды) предназначены для отображения информации в виде блока с рамкой и закругленными углами.
Для типовых элементов заготовлены классы семейства .tui-island:
  • .tui-island__category: категория
  • .tui-island__title: заголовок
  • .tui-island__paragraph: абзац текста
    • .tui-island__paragraph_link: абзац с ссылками. Отступ вверх 16px
    • .tui-island__paragraph_button: абзац с кнопками. Отступ вверх 20px
  • .tui-island__figure: фигура
  • .tui-island__content: второстепенный класс для случая с фигурой сбоку
  • .tui-island__carousel-controls: блок с элементами управления каруселью
  • .tui-island__carousel-button: кнопка управления каруселью
  • .tui-island__carousel-pages: страницы карусели
  • .tui-island__footer-button: футерная кнопка на всю ширину айленда
