Компонент календаря для мобильных устройств. Используется автоматически в InputDate и InputDateRange при подключении модуля TuiMobileCalendarDialogModule.

Для передачи текущего значения используется провайдер токен TUI_CALENDAR_DATA_STREAM, см. пример.

Компонент рассчитан на полноэкранное использование на мобильных устройствах. Функция, определяющая, заблокирована ли дата в календаре.
Должна быть чистой функцией. Максимальная дата Минимальная дата Одиночная дата или диапазон Нажатие на крестик Нажатие кнопки "Готово"
